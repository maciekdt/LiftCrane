# LiftCrane
Simple serverless CMMS for lift company. System powered by Google Firebase.  

Tools used :
  - Kotlin for android app
  - Vue.js for web app
  - Firebase firestore 
  - Firebase auth service
  - Firebase hosting
  - Firebase storage
